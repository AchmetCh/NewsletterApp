# Newsletter Drag and Drop App

A powerful newsletter creation tool built with the MERN stack, featuring a sophisticated drag-and-drop email editor fueled by [react-email-editor](https://www.npmjs.com/package/react-email-editor).

## 🚀 Features

- **Drag & Drop Editor**: Intuitive interface for creating beautiful emails without coding, using `react-email-editor`.
- **Responsive Design**: Emails look great on all devices.
- **Template Management**: Save and load your email designs.
- **MERN Stack**: Robust full-stack architecture.

## 🛠️ Tech Stack

- **Frontend**: React.js
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Editor**: react-email-editor (Unlayer)

## 📦 Installation

1.  **Clone the repository**
    ```bash
    git clone <repository-url>
    cd <repository-directory>
    ```

2.  **Install Server Dependencies**
    ```bash
    cd server
    npm install
    ```

3.  **Install Client Dependencies**
    ```bash
    # Assuming client is in the root or a separate 'client' folder, adjust as needed.
    # If client is in root:
    npm install
    ```
    *(Note: Adjust the folder paths based on your actual project structure)*

4.  **Configuration**
    - Create a `.env` file in the server directory and configure your MongoDB URI and port.

5.  **Run the Application**
    - Start the server: `npm start` (in server directory)
    - Start the client: `npm start` (in client directory)
